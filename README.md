## @simon_he/git-fork
提供一个跳转到GitHub地址的小组件

## 灵感来源
- [github-corners](https://tholman.com/github-corners/)
- [github-ribbons](https://github.com/superhugo/github-ribbons)

## 参数
```typescript
export interface Props {
  link: string ## GitHub地址
  position: 'left' | 'right' ## 组件位置
  content: string ## 组件内容
  color: string ## 组件颜色
  type: 'corners' | 'ribbons' ## 提供2种类型的组件
}
```
